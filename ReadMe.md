# Mechanistic Interpretability by Design

This repository contains the website for the tutorial **"Neural Networks meet Explicit Knowledge Representation: Towards Mechanistic Interpretability and Neuro-symbolic Modeling by-design."**

The tutorial explores the intersection of **Transformer-based Large Language Models (LLMs)**, **Mechanistic Interpretability**, **Neuro-symbolic AI**, **Explicit Knowledge Representation** and **Interpretability by Design**. It introduces approaches for understanding and analyzing the internal mechanisms of neural language models and discusses how structured and symbolic representations can contribute to more interpretable and transparent AI systems.

The website is available at:

**https://humancentricart.github.io/mechanistic-interpretability-by-design/**

## Tutorial Editions

The website currently provides two conference-specific editions of the tutorial:

* **ISWC 2026**
* **EKAW 2026**

Each edition provides the relevant tutorial information for the corresponding venue, including the tutorial description, intended audience, content, program, and presenter information.

## Website Structure

The repository is organized as a lightweight static website built with **HTML, CSS, and JavaScript**.

The main components include:

* `index.html` — the main entry point, where visitors can select the tutorial edition.
* `iswc/` — content and pages specific to the **ISWC 2026** edition.
* `ekaw/` — content and pages specific to the **EKAW 2026** edition.
* `components/` — reusable HTML components used across the website.
* `js/` — JavaScript modules supporting navigation, component loading, interactive elements, and other website functionality.
* `Assets/` — images and other visual assets used by the website.
* `style.css` — shared styling for the website.
* `presenters.html` — presenter information.
* `other-versions.html` — information about other available editions or versions of the tutorial.

## Tutorial Topics

The tutorial introduces and discusses topics including:

* Transformer-based Large Language Models interpretability
* Mechanistic Interpretability
* Neuro-symbolic AI
* Explicit Knowledge Representation
* The relationship between neural and symbolic reasoning
* Memorization and generalization in language models
* Interpretable and transparent model architectures
* Designing neural architectures with interpretability in mind

The tutorial is intended to provide an accessible introduction to these topics, with intuitive explanations preceding more technical material and equations.

## Website Development

The website is a static academic tutorial page and does not require a backend server or database. It can be deployed directly using static hosting services such as **GitHub Pages**.

The site is implemented using standard web technologies:

* **HTML** for content and structure
* **CSS** for styling and layout
* **JavaScript** for navigation, reusable components, and interactive functionality
